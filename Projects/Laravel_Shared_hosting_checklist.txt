======= File Upload ============
1. Make zip of the project folder
2. cPanel > File Manager > public_html > upload
3. Extract the zip file
3. Settings > Show hidden files
5. Move contents of Project > public folder to public_html
6. Edit index.php (replace /.../ with ProjectFolderName)
7. Permission(777) to storage folder
8. bootstrap > cache > delete all
9. cPanel > multiPHP manager > apply php 8
10. edit .env file
